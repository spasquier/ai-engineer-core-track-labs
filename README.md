# ai-engineer-core-track-labs
Jupyter Notebook labs for AI Engineer Core Track course

## Installing dependencies
Install `uv`
https://docs.astral.sh/uv/getting-started/installation/

Execute `uv sync`

You can use `pip install -r requirements.txt` but it's much more slower.

## Setting up environment
You need to create a `.env` file in the root of the project.

It should contain you OpenAI API Key, Google API Key and HuggingFace secret token
```
OPENAI_API_KEY=
GOOGLE_API_KEY=
HF_TOKEN=
```
